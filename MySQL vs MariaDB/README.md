# MySQL vs MariaDB

> 차이가 뭘까?
>
> - Oracle에서 MySQL을 인수한 것에 반대한 직원들이 나와서 만든 DB가 MariaDB
> - MySQL은 사용자를 늘리기 위해 무료화
>
> OracleJDK의 유료화
>
> OpenJDK는 무료

MySQL이 언제 유료화될지 모르기 때문에

최근 기업들이 MySQL -> MariaDB로 바꾸기 시작

예) 카카오뱅크

- MySQL과 MariaDB 모두 같은 driver를 사용하기 때문에 사용에 있어서 다른 것은 없음!
